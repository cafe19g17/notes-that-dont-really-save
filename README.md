<h1>Notes That Don't Really Save</h1>

<p>"Notes that don't really save", é um único arquivo em Python que simula um editor de notas. Você pode criar notas, apagar notas e ler todas as suas notas criadas, mas como o próprio nome já diz, elas não salvam quando você para de executar. O motivo é simples: eu só quis criar um simulador de anotações. </p>
